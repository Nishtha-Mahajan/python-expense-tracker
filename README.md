# Python Expense Tracker

This is a simple console-based Expense Tracker project built using Python.

## Features
- Add daily expenses
- View all expenses
- View total expenses
- Menu-driven program

## Concepts Used
- Python Lists
- Dictionaries
- Loops
- Conditional Statements
- User Input Handling

## How to Run
1. Install Python on your system
2. Run the file using:
   python MiniProject.py
